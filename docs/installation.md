# Installation Guide

## Prerequisites

- Node.js >= 14.x
- npm >= 6.x

## Global Installation

```bash
npm install -g mcp-security-scanner
```

## Local Installation

```bash
npm install mcp-security-scanner
```

## Verifying Installation

```bash
mcp-scan --version
```
